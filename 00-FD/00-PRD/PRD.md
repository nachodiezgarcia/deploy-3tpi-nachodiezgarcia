# INTRO

Voy a hacer un portal de tipo academia online, en la que el alumno se loguee, vea unos cursos online a los que tiene acceso, y pueda ver las lecciones, así como la guía de cada una.

# DESAFÍOS

* **Tengo que proveer de recursos estáticos**, tipo vídeo y guías en Markdown.
* **Tengo que dar una manera fácil** para poder subir estos recursos y que un usuario no técnico pueda actualizar el portal.
* **Quiero ahorrar implementación de intranet** lo máximo posible.
* **Quiero que este material esté en un sitio seguro y rápido** (storage + CDN).
* **Quiero que un usuario tenga que autenticarse** para poder acceder.
* **Quiero que un usuario tenga acceso a ciertos cursos.**
* **Quiero guardar el progreso** del usuario.
* **Necesito una intranet** para gestionar usuarios y cursos.
